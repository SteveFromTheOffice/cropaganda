## Project Context

- **Project:** cropaganda — Windows desktop app for fast photo cropping to 4:5 (Instagram format)
- **Stack:** C# / WPF, Windows
- **Owner:** Michael Scott
- **Created:** 2026-04-13

## Features

- Drag-and-drop batch photo import
- 4:5 crop overlay with mouse-wheel zoom, mouse drag to pan
- Enter key to confirm and advance to next image
- Output written to a folder as user progresses
